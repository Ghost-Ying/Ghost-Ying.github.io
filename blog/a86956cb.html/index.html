Cisco IOS 基础命令集汇总

发表于2022-11-30|更新于2022-12-06|思科CCNA网络基础入门
|字数总计:1.1k|阅读时长:3分钟|阅读量:

一：Cisco IOS

1.1 简介

  • Cisco internetwork operating system（IOS）：为 Cisco 设备配备的系统软件。应用于路由器、局域网交换机、小型无线接入点、具有几十个接口的大型路由器等。

1.2 功能

  • Cisco IOS 可为设备提供以下网络服务
    • 基本的路由和交换功能
    • 安全可靠地访问网络资源
    • 网络可伸缩性

1.3 访问方式

  • 可以通过多种方法访问 CLI 环境，常见方式：
    • 控制台
    • Telnet 或者 SSH
    • 辅助端口

1.4 访问模式

  • 用户执行模式 “>”
  • 特权执行模式 “#”
  • 全局配置模式
    • 从全局配置模式可进入多种不同的配置模式。其中每种模式可用于配置 IOS 设备的特定部分或者特定功能。
  • 其他特定模式
    • 接口模式：用于配置一个网络接口（Fa0/0、S0/0/0 等）
    • 线路模式：用于配置一条线路（实际线路或者虚拟线路），如控制台、AUX、VTY等
    • 路由配置模式：用于配置一个路由选择协议进程的相关参数

1.5 模式切换

  • enable：用户执行模式进入特权模式
  • configure terminal：特权模式进入全局配置模式
  • exit：返回上一级模式
  • end：直接返回特权模式

1.6 基础操作

  • ?：使用 ? 可以匹配查看命令库
  • 设置IP：ip address IP地址 子网掩码
  • no：使用 no shutdown，退出保持状态开启。
  • show running-config：查看配置信息
  • show ip interface brief：查看 ip 简短信息
  • hostname 名称：设置主机名称（全局模式下）
  • enable password 密码：设置主机密码
  • no enable password：关闭密码验证
  • enable secret 密码：采用加密方式设置密码

1.7 语法检查

  • incomplete command：命令不完整
  • invalid input detected at ‘^’ marker：箭头所指字符无法识别
  • ambiguous comm：未知输入

1.8 基本命令结构

  • 每个 IOS 命令都具有特定的格式或者语法，并在相应的提示符下执行
  • 常规命令语法为命令后接相应的关键字和参数
    • Router> ping 192.168.1.2
  • 某些命令包括一个关键字和参数子集，此子集可提供额外功能

1.9 快捷键

  • Tab：填写命令或者关键字的剩下部分
  • Ctrl+R：重新显示一行
  • Ctrl+Z：退出配置模式并返回执行模式
  • Ctrl+C：放弃当前命令并退出配置模式
  • Ctrl+Shift+6：用于中断诸如 ping 或 traceroute 之类的 IOS 进程
  • 向上箭头：上一条键入命令
  • 向下箭头：下一条键入命令

1.10 常用配置

  • 配置用户登录密码
    • 控制台口令：用于限制人员通过控制台连接访问设备
    • 使能口令：用于限制人员访问特权执行模式
    • 使能加密口令：经加密，，用于限制人员访问特权执行模式
    • VTY口令：用于限制人员通过 Telnet 访问设备
  • 管理配置文件
    • 将当前配置写入启动配置文件
      • write
      • copy running-config startup-config
    • 删除启动配置 文件
      • erase startup-config
      • delete flash:config.text
  • 接口配置
    • 进入接口
      • interface 接口类型 插槽/接口编号
    • 为接口配置IP地址
      • ip address ip地址 子网掩码
    • 激活接口
      • no shutdown（默认是shutdown状态）
  • show 命令
    • show version：查看当前操作系统版本
    • show running-config：查看运行配置
    • show startup-config：查看启动配置
    • show flash：查看FLASH
    • show cpu：查看cpu利用率
    • show memory：查看内存使用情况
    • show interface：查看端口
  • 验证配置
    • ping
    • traceroute

二：以太网技术线缆标准

  • T-568A：绿白-绿-橙白-蓝-蓝白-橙-棕白-棕
  • T-568B：橙白-橙-绿白-蓝-蓝白-绿-棕白-棕
  • 标准网线：不同设备用于直通线，即两头都是568A或者都是568B
  • 交叉网线：同等设备用交叉线即一头为568A，一头为568B
  • 全反线（一般用于配置线缆）两端线序刚好相反

三：工程三招

  • no ip domain lookup：关闭域名解析（防止敲错命令查询）
  • line console 0
    • logging synchronous：log当前显示信息同步
    • no exec-timeout：关闭会话超时，防止一段时间不操作自动退出
